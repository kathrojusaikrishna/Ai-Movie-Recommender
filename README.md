# 🎬 AI Movie Recommendation System

##  Overview
This project builds a personalized movie recommendation system using machine learning techniques like content-based and collaborative filtering.

##  Dataset
- MovieLens / TMDB dataset
- Includes movie metadata (genres, cast, overview)

##  Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Cosine Similarity / KNN

## How it Works

1. Data Cleaning & Preprocessing
2. Feature Engineering (TF-IDF / CountVectorizer)
3. Similarity Calculation (Cosine Similarity)
4. Recommendation Generation

##  Example Output
Input: Avatar  
Output:
- Guardians of the Galaxy  
- John Carter  
- Star Trek  

##  Features
- Fast recommendation using vectorized similarity
- Scalable logic for large datasets
- Modular ML pipeline
